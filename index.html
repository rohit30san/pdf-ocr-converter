Browser-based OCR

Turn PDFs and images into clean text.

Upload a PDF or image, choose an OCR language, define a page range, and extract text entirely in the browser with PDF.js and Tesseract.js.

No backend Drag and drop Per-page results Searchable PDF output

How It Works

Select a file, pick the OCR language, and run extraction. PDFs are rendered page by page in-browser, then passed to Tesseract for recognition.

Tip: first-time OCR for a language may take longer because trained data is downloaded by the browser.

This page is running from file://, which breaks browser OCR workflows in many browsers. Serve this folder over local HTTP instead, for example: python -m http.server 8000 and then open http://localhost:8000.
